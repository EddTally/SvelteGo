# SvelteGo
Testing a Golang backend with SvelteKit frontend

To get it up and working:

In a terminal run this to get the gin server started  
> cd web-service-gin  
> go mod tidy  
> go run .  

In a seperate terminal run this to start the website  
> cd ../sveltekit  
> npm install  
> npm run dev
